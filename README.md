# Grape simplest API

This is a Grape API using Rack and MiniTest.


# Install

```bash
$ gem update
$ bundle install
```

## Usage

#### Run tests:

```bash
$ rake
```

#### Run server:

```bash
$ rackup
```
Then go to <http://localhost:9292/api/v0/hello>

View your swagger documentation on <http://localhost:9292/api/v0/swagger_doc>
